# EndTerm-Project
Personal Finance Tracker

A simple web app to track income, expenses, balance, and personal debts (Udhar).
Built using HTML, CSS, and Vanilla JavaScript. Data is stored locally in the browser.

Live Demo:
https://kushagra-923.github.io/EndTerm-Project/

# Features

Add income & expenses

Auto balance calculation

Filter transactions

Category-wise spending

Udhar (debt) tracking

Works offline (LocalStorage)

# Tech Stack

HTML

CSS

JavaScript

# Run Locally

Just open index.html in your browser.
